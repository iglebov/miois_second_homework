# miois_second_homework

## Установка необходимых зависимостей

1. Перейти в файл requirements.jl
2. Выполнить (при помощи Julia) строки кода указанные в файле

## Запуск кода

1. Открыть терминал
2. Перейти в директорию miois_second_homework
3. Запустить файл с экспериментами:
```bash
julia --threads=auto --project=. src/experiment/efficiency.jl
```
